b量子速读 - 使用说明

从B站视频提取字幕/评论，整理为结构化 Markdown

📋 功能简介

  • 提取字幕 — 从 B站 视频提取 CC 字幕和评论，生成 Markdown 文件
  • 语音识别 — 对没有字幕的视频，通过本地 Whisper 服务进行语音转文字

🎤 安装 Whisper 语音识别服务

语音识别功能需要在本地运行 Whisper 服务。有两种方式：一键安装（推荐）和手动安装。

✅ 方式一：一键安装（推荐）

  1. 打开扩展窗口，点击「🔧 安装向导」按钮（Whisper 未连接时显示）
  2. 安装向导自动填入扩展 ID，点击「📋 复制安装命令」
  3. 打开文件管理器，进入 whisper_server 目录
  4. 在目录地址栏输入 cmd 回车，粘贴上一步复制的命令并回车
  5. 安装脚本自动完成：创建虚拟环境 → 安装依赖(清华镜像) → 预下载模型(HF镜像) → 检测 ffmpeg → 注册本地服务
  6. 安装完成后点「🔄 检查安装状态」，确认全部 ✅
  7. 重启 Chrome 浏览器

🚀 一键启动

  1. 打开扩展窗口，点击「🚀 一键启动」
  2. 首次启动自动下载模型（约 500MB），扩展窗口显示下载进度
  3. 看到绿色圆点 ● Whisper: 已连接，即可使用语音识别
💡 你也可以直接双击 whisper_server/start_server.bat 启动服务（无需安装向导）。

⚙️ 高级：手动安装

点击展开手动安装步骤

步骤 1：检查 Python 环境

需要 Python 3.8 或更高版本。打开终端（CMD/PowerShell）验证：

python --version

如果未安装，请从 python.org 下载安装。
安装时请勾选 "Add Python to PATH"。

步骤 2：创建虚拟环境并安装依赖

cd whisper_server python -m venv .venv .venv\Scripts\pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple

步骤 3：配置模型（可选）

可通过环境变量指定模型参数：

变量说明默认值推荐值
WHISPER_MODEL 模型大小（越大越准越慢） small 有 GPU: large-v3；无 GPU: base
WHISPER_DEVICE 计算设备 cpu 有 NVIDIA GPU: cuda
WHISPER_COMPUTE_TYPE 计算精度 int8 GPU 推荐: float16
WHISPER_PORT 服务端口 8787 一般不需要修改（新版可自动找空闲端口）

配置示例（Windows PowerShell）：

$env:WHISPER_MODEL="base" $env:WHISPER_DEVICE="cpu" .venv\Scripts\python server.py

步骤 4：预下载模型（可选但推荐）

set HF_ENDPOINT=https://hf-mirror.com .venv\Scripts\python -c "from faster_whisper import WhisperModel; WhisperModel('small', device='cpu', compute_type='int8')"

步骤 5：启动服务

.venv\Scripts\python server.py

看到以下输出即表示启动成功：

[INFO] Model loaded successfully * Running on http://0.0.0.0:8787

📄 查看日志

所有日志文件位于 whisper_server/logs/：

  • server.log — 服务主日志（轮转，保留最近 3 份）
  • server_stdout.log / server_stderr.log — 启动脚本输出
  • native_host.log — 原生消息主机日志（一键启动的诊断信息）

🔧 故障排查

  • 一键启动按钮没反应？
    先运行安装向导，确认 5 项全部 ✅。如仍然失败，运行 install.bat -Repair 修复安装。
  • 模型下载非常慢怎么办？
    新版服务已自动设置 HF 国内镜像（https://hf-mirror.com），无需手动配置。如仍慢，检查网络环境。
  • 端口被占用？
    新版服务可自动寻找空闲端口（8788、8789...），扩展也会自动同步，无需手动处理。
  • 电脑没有 GPU / 内存不足？
    使用 tiny 或 base 模型，资源占用更低，速度快。
  • CPU 模式下识别速度慢？
    这是正常的。small 模型处理 10 分钟音频约需 3-5 分钟。建议使用 base 模型提升速度。
  • 语音识别中断 / 崩溃？
    新版服务有自动重启功能，如崩溃会尝试自动恢复。查看 logs/server.log 获取诊断信息。
  • 需要一直保持扩展窗口打开吗？
    不需要。新版支持后台运行（offscreen 模式），关闭窗口后转录仍会继续。重新打开扩展可查看结果。

💡 使用提示

  • 语音识别前请确保网络畅通（服务需要下载 B站 音频流）
  • 长视频（超过 30 分钟）的转写可能需要一些时间，进度条显示真实进度
  • 取消按钮可随时中断正在进行的转写
  • 转写完成后会弹出系统通知并伴有提示音，鼠标移动到扩展窗口上或点击通知可停止铃声
