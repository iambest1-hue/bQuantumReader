b量子阅读 - 使用说明

从B站视频提取字幕/评论，整理为结构化 Markdown

📋 功能简介

  • 提取字幕 — 从 B站 视频提取 CC 字幕和评论，生成 Markdown 文件
  • 语音识别 — 对没有字幕的视频，通过本地 Whisper 服务进行语音转文字

🎤 安装 Whisper 语音识别服务

语音识别功能需要在本地运行 Whisper 服务。首次配置后，后续使用只需启动服务即可。

步骤 1：检查 Python 环境

需要 Python 3.8 或更高版本。打开终端（CMD/PowerShell）验证：

python --version

如果未安装，请从 python.org 下载安装。
安装时请勾选 "Add Python to PATH"。

步骤 2：安装依赖包

在终端中进入扩展目录下的 whisper_server 文件夹：

cd 扩展目录/whisper_server pip install -r requirements.txt
💡 如果安装速度慢，可使用国内镜像源：
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple

步骤 3：配置模型（可选）

可通过环境变量指定模型参数：

变量说明默认值推荐值
WHISPER_MODEL 模型大小（越大越准越慢） small 有 GPU: large-v3；无 GPU: base
WHISPER_DEVICE 计算设备 cpu 有 NVIDIA GPU: cuda
WHISPER_COMPUTE_TYPE 计算精度 int8 GPU 推荐: float16
WHISPER_PORT 服务端口 8787 一般不需要修改

配置示例（Windows PowerShell）：

$env:WHISPER_MODEL="base" $env:WHISPER_DEVICE="cpu" python server.py

模型下载说明：faster-whisper 会在首次运行时自动下载模型文件（small 约 500MB，base 约 150MB）。下载速度取决于网络，也可以设置镜像加速：

$env:HF_ENDPOINT="https://hf-mirror.com" python server.py

步骤 4：启动服务

python server.py

首次启动时会自动下载模型，请耐心等待。看到以下输出即表示启动成功：

B站学神 Whisper Server starting on port 8787... Pre-loading model... (this may take a while on first run) Model loaded successfully * Running on http://0.0.0.0:8787

步骤 5：测试连接

服务启动后，在浏览器中访问：

http://localhost:8787/status

返回类似 {"ready":true,"model":"small","device":"cpu"} 即表示服务正常运行。

然后在扩展 设置页面 中确认 Whisper 服务地址为 http://localhost:8787，点击"测试连接"。

❓ 常见问题

  • 模型下载非常慢怎么办？
    设置环境变量 HF_ENDPOINT=https://hf-mirror.com 使用 HuggingFace 国内镜像加速下载。
  • 端口被占用如何解决？
    修改端口：$env:WHISPER_PORT="8788"，然后在扩展设置中同步修改服务地址。
  • 电脑没有 GPU / 内存不足？
    使用 tiny 或 base 模型，资源占用更低，速度快。
  • CPU 模式下识别速度慢？
    这是正常的。small 模型处理 10 分钟音频约需 3-5 分钟。建议使用 base 模型提升速度。
  • Windows 路径包含空格？
    请用引号包裹路径，例如 cd "D:\my extensions\b站学神\whisper_server"。
  • 如何自动下载模型而不启动服务？
    运行以下 Python 代码即可预下载：python -c "from faster_whisper import WhisperModel; WhisperModel('small')"

⚠️ 重要提醒

在 提取字幕 或 语音识别 过程中，请务必保持扩展窗口打开，不要关闭！

如果窗口关闭，正在进行的操作将中断。语音识别完成后，结果会自动显示在窗口中。
